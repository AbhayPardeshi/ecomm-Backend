## **How to install and run locally ?**

```
$ git clone https://github.com/rdshinde/swadeshi-backend.git
$ npm install
$ npm start
```

## **Features -**

- Create Users.
- Create Categories.
- Create Products.

- User Cart Management
  - Add to Cart.
  - Update Cart.
  - Move to wishlist from Cart.
  
- User Wishlist Management
  - Add to Wishlist
  - Move to cart from Wishlist
  - Delete from Wishlist
  
- Products Management
  - Add new products
  - Fetch all products
  - Fetch Single products

- Manage User Address.
  - Add new address
  - Edit address
  - Delete address
  
- Authentication
  - User Signup
  - User Login

## **Built with -**

- Node JS
- Express JS
- Mongo DB Atlas
- Mongoose
